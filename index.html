What is Python?

Python is a simple, readable programming language used for web development, data analysis, automation, and more. It’s beginner-friendly and widely used.

Setting Up Python

Download Python from python.org and install it. Use an editor like VS Code or IDLE to write code.

To check if Python is installed, open a terminal and type:

python --version

Run a Python file with:

python filename.py

Variables

Variables store data like numbers or text. You don’t need to declare their type.

name = "Alice"  # String
age = 25        # Integer
price = 19.99   # Float
is_student = True  # Boolean

Rules: Use lowercase, no spaces, start with a letter or underscore.

Data Types

Python has several data types to store different kinds of data:

  • int — Whole numbers (e.g., 5)
  • float — Decimal numbers (e.g., 3.14)
  • str — Text (e.g., "Hello")
  • bool — True or False
  • list — Ordered, mutable collection
  • tuple — Ordered, immutable collection
  • dict — Key-value pairs
  • set — Unordered, unique items

Check type with:

x = 42
print(type(x))  # Output:

Operators

Operators perform operations on data.

  • Arithmetic: +, -, *, /, % (modulus), ** (power)
  • Comparison: ==, !=, >, <, =, <=
  • Logical: and, or, not
a = 10
b = 3
print(a + b)  # Output: 13
print(a > b)  # Output: True
print(a % b)  # Output: 1

Conditionals

Use if, elif, and else to make decisions.

age = 18
if age >= 18:
    print("Adult")
elif age >= 13:
    print("Teen")
else:
    print("Child")

Indentation (spaces) is important in Python!

Loops

Loops repeat code. Use for for a set range or while for conditions.

# For loop
for i in range(5):
    print(i)  # Output: 0, 1, 2, 3, 4

# While loop
count = 0
while count < 3:
    print(count)
    count += 1  # Output: 0, 1, 2

Functions

Functions are reusable blocks of code.

def greet(name):
    return f"Hello, {name}!"

print(greet("Alice"))  # Output: Hello, Alice!

Use def to define a function and return to send back a value.

Lists

Lists store multiple items in order. They are mutable (changeable).

fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # Output: apple
fruits.append("orange")  # Add item
print(fruits)  # Output: ['apple', 'banana', 'cherry', 'orange']

Common methods: append(), remove(), pop(), sort().

Dictionaries

Dictionaries store key-value pairs.

person = {"name": "Alice", "age": 25}
print(person["name"])  # Output: Alice
person["city"] = "New York"  # Add key-value
print(person)  # Output: {'name': 'Alice', 'age': 25, 'city': 'New York'}

Tuples

Tuples are like lists but immutable (cannot change).

coords = (10, 20)
print(coords[0])  # Output: 10

Sets

Sets store unique items, unordered.

numbers = {1, 2, 2, 3}
print(numbers)  # Output: {1, 2, 3}

Strings

Strings are text, enclosed in quotes.

text = "Hello, World!"
print(text.upper())  # Output: HELLO, WORLD!
print(text[0:5])  # Output: Hello

Common methods: upper(), lower(), strip(), split().

File Input/Output

Read from or write to files.

# Write to a file
with open("example.txt", "w") as file:
    file.write("Hello, Python!")

# Read from a file
with open("example.txt", "r") as file:
    content = file.read()
print(content)  # Output: Hello, Python!

Modules

Modules are pre-written code you can use.

import math
print(math.sqrt(16))  # Output: 4.0

Install external modules with pip install module_name.

Exception Handling

Handle errors gracefully with try and except.

try:
    x = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero!")

Object-Oriented Programming (OOP)

OOP organizes code using classes and objects. Think of a class as a blueprint and an object as something built from that blueprint.

Key concepts:

  • Class: Defines properties (attributes) and behaviors (methods).
  • Object: An instance of a class.
  • Attributes: Data stored in an object.
  • Methods: Functions inside a class.
  • Inheritance: A class can inherit from another class to reuse code.
  • Encapsulation: Hide data using private attributes (with _ or __).
  • Polymorphism: Same method works differently for different classes.

Example: Create a Dog class and use it.

# Define a class
class Dog:
    def __init__(self, name, age):  # Constructor to initialize attributes
        self.name = name  # Attribute
        self.age = age

    def bark(self):  # Method
        return f"{self.name} says Woof!"

# Create objects (instances)
dog1 = Dog("Buddy", 3)
dog2 = Dog("Max", 5)

# Access attributes and methods
print(dog1.name)  # Output: Buddy
print(dog1.bark())  # Output: Buddy says Woof!
print(dog2.age)  # Output: 5

Example: Inheritance

class Animal:
    def __init__(self, species):
        self.species = species

    def make_sound(self):
        return "Some sound"

class Cat(Animal):  # Cat inherits from Animal
    def __init__(self, name, species):
        super().__init__(species)  # Call parent class constructor
        self.name = name

    def make_sound(self):  # Override parent method
        return f"{self.name} says Meow!"

cat = Cat("Whiskers", "Feline")
print(cat.species)  # Output: Feline
print(cat.make_sound())  # Output: Whiskers says Meow!

Tips: Use self to refer to the object’s attributes/methods. Use super() for inheritance.

Best Practices

Write clean, readable Python code.

  • Use meaningful variable and class names.
  • Add comments to explain code.
  • Follow PEP 8 style guidelines.
  • Keep functions and methods short and focused.
  • Test your code regularly.
  • Use encapsulation to protect class data.
